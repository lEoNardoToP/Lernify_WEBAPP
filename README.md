# Lernify_WEBAPP

Lernify is a learning-oriented web application prototype.

The goal of this repository is to show a practical web app structure and evolve it into a cleaner portfolio project with better documentation, setup instructions, screenshots, and a clearer product direction.

---

## Why this project matters

This project is useful as portfolio material because it can demonstrate:

- web application structure
- frontend/backend integration
- user-facing product thinking
- learning-platform features
- iterative improvement from prototype to polished project

---

## Planned product direction

Lernify can become a small learning dashboard where users can:

- organize study material
- track progress
- manage topics and lessons
- review exercises or notes
- use AI-assisted summaries or study recommendations

---

## Current status

**Status:** prototype / portfolio cleanup in progress.

The repository is being improved to make it easier for recruiters and developers to understand:

- what the app does
- how it is structured
- how to run it locally
- what features are implemented
- what should be improved next
